# 综述：人工智能与社会（review_latest）

人类社会已经迈进以计算、数据与算法为关键基础设施的数字社会。人工智能不再仅是一组面向特定任务的工程方法，而逐渐演化为组织知识生产、经济流程与社会互动的通用能力框架。

## 一、从“研究对象”到“研究装置”：LLM 进入社会科学的方法链条

数字社会科学的研究对象与研究装置正在同步变化。LLM 既被设想为模拟主体与替代受访者，也被当作文本劳动的技术工具。核心张力在于：LLM 显著扩展了社会模拟与文本处理的规模与速度，但其对齐机制与语料学习带来的偏差、同质化倾向及不可见的中介效应，可能在不易察觉处改写解释权的分配。

## 二、理论视角：关系潜能、工具链纠缠与方法论反身性

相关研究借助关系与物质社会学，讨论 LLM 在具体制度与组织工具链条中的“关系潜能”。方法论反身性与研究者责任被置于中心位置；LLM 被视为一种解释性技术，纳入研究并非中性的技术选择，而会改变责任归属、可追溯性与可问责性。

## 三、相对稳定的结论

- LLM 正被持续设想为“类人研究对象/代理”，但其有效性与适用边界需要以社会科学的规范与标准加以约束。
- 反身性被多次强调：将 LLM 纳入研究会重塑解释过程，改变责任如何被分配、证据如何被记录与追溯。
- 生成式 AI 的扩散被描述为数据质量的结构性挑战，迫使研究者重新讨论何为“自然语言的原生态表达”。
